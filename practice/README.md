# Thư mục thực hành & Code Lab (Hands-on Practice)

Đây là nơi lưu trữ toàn bộ mã nguồn thực tế (Terraform, YAML manifest, Dockerfile, v.v.) được viết trong quá trình học và thực hành.

## Cấu trúc thư mục
* `terraform/`: Lưu trữ các file cấu hình Terraform thực hành.
* `kubernetes/`: Lưu trữ các file YAML manifest (Pod, Service, ConfigMap...) thực hành.
* `w8-lab/`: Lưu trữ toàn bộ mã nguồn, cấu hình của bài **Lab W8 (Mini K8s platform trên minikube)**.
